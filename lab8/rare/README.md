# Финансовый трекер расходов

## Описание

Приложение для отслеживания личных расходов с графическим интерфейсом на PyQt6. Позволяет добавлять, просматривать и удалять расходы, фильтровать их по категориям и визуализировать данные в виде диаграмм.

---

## Инструкции по запуску

### Установка зависимостей

```bash
py -m pip install PyQt6 PyQt6-Charts
```

### Запуск

```bash
py tracker.py
```

---

## Краткая справка

### Вкладка «Расходы»

| Элемент | Описание |
|---|---|
| Дата | Выбор даты расхода через календарь |
| Категория | Еда, Транспорт, Жильё, Развлечения, Здоровье, Одежда, Прочее |
| Сумма | Сумма расхода в рублях |
| Описание | Произвольный комментарий (необязательно) |
| Кнопка «Добавить» | Сохранить расход |
| Фильтр | Отображать расходы только выбранной категории |
| Кнопка «Удалить выбранное» | Удалить выделенную строку |

Итоговая сумма всех расходов отображается в заголовке окна.

### Вкладка «Графики»

| Элемент | Описание |
|---|---|
| Круговая диаграмма | Доли расходов по категориям |
| Столбчатая диаграмма | Сравнение сумм по категориям |
| Кнопка «Обновить графики» | Перестроить диаграммы по текущим данным |

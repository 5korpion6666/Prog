# Финансовый трекер расходов

## Описание

Приложение для отслеживания личных расходов с графическим интерфейсом на PyQt6 и хранением данных в SQLite. Позволяет добавлять, просматривать и удалять расходы, фильтровать их по категориям, visualизировать данные в виде диаграмм и просматривать статистику по месяцам.

---

## Инструкции по запуску

### Установка зависимостей

```bash
py -m pip install PyQt6 PyQt6-Charts
```

### Запуск

```bash
py Tracker_medium.py
```

База данных `tracker.db` создаётся автоматически в той же папке, что и файл программы. Данные сохраняются между запусками.

---

## Краткая справка

### Вкладка «Расходы»

| Элемент | Описание |
|---|---|
| Дата | Выбор даты расхода через календарь |
| Категория | Еда, Транспорт, Жильё, Развлечения, Здоровье, Одежда, Прочее |
| Сумма | Сумма расхода в рублях |
| Описание | Произвольный комментарий (необязательно) |
| Кнопка «Добавить» | Сохранить расход в БД |
| Фильтр | Отображать расходы только выбранной категории |
| Кнопка «Удалить выбранное» | Удалить выделенную строку из БД |

Итоговая сумма всех расходов отображается в заголовке окна.

### Вкладка «Графики»

| Элемент | Описание |
|---|---|
| Круговая диаграмма | Доли расходов по категориям |
| Столбчатая диаграмма | Сравнение сумм по категориям |
| Кнопка «Обновить графики» | Перестроить диаграммы по текущим данным |

### Вкладка «Статистика»

| Элемент | Описание |
|---|---|
| Таблица | Категории, суммы и доли расходов в процентах |
| График по месяцам | Столбчатая диаграмма расходов по месяцам |
| Кнопка «Обновить статистику» | Обновить данные |

### Интеграция с БД

Данные хранятся в SQLite (`tracker.db`). Структура таблицы:

```sql
CREATE TABLE expenses (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    date        TEXT    NOT NULL,
    category    TEXT    NOT NULL,
    amount      REAL    NOT NULL,
    description TEXT    DEFAULT ''
)
```
